Transitions

May 23th, 2013

Experience the full transitions on this page through a desktop browser. Mouse over them to start the magic!

If last week's lesson was transformations, then this week has to be transitions and animations. Some of these consist of only transitions while other have animate built into them. As I create more examples, I will try to mix up the variety and include JavaScript.

As I am coding this page up, I am beginning to realize how bulky @keyframes really is. I had originally though Compass would support @include keyframes but I was greatly disappointed when I found they got nothing. However, my instructor claimed, last week, that there the web have great remedies around this major problem. If I more resources/remedies, I will update this page. In the spare time, please check out some of my experimentation. Remember this is a work in progress.
